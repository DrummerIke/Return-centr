G7 Return Center — premium v3

Что сделано:
- отслеживание читает данные из той же таблицы
- свежие возвраты всегда выводятся сверху
- форма упрощена:
  - статус возврата убран из формы
  - e-mail убран из формы
  - сумма возврата показывается только при частичном возврате
  - позиции к возврату показываются только при частичном возврате
- инструкции остаются отдельным нижним разделом

Что делать:
1. Вставить ID таблицы в apps-script.gs
2. Проверить SHEET_NAME
3. Развернуть Apps Script как Web App
4. Вставить URL Web App в CONFIG.APPS_SCRIPT_URL в index.html
5. Залить index.html на GitHub Pages
6. Встроить страницу в Google Sites
